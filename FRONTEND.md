### Вакансия Frontend React Developer в компанию АНО Университет 2035

К нам в команду 2035.university требуется frontend react разработчик в отдел разработки инструментов анализа и визуализации данных рынка труда.
Мы делаем крутые сервисы, которые помогают сделать рынок труда более прозрачным. Нам нужен специалист который сможет собрать фронт на Ant Design, настроить роутинг react-router, стилизовать его с помощью scss/styled-components, и по необходимости подключить Yandex Map API и визуализировать данные с помощью Ant Charts.

**Ты подойдешь, если умеешь:**
- React/Typescript
- Redux, React Context
- scss/styled-components

**Плюсом будет**
- Опыт работы с Gitlab/Gitlab CI
- Jira
- git

**Требования:**
- Гражданство РФ

**Предлагаем:**
- Официальное оформление
- Зарплатная вилка 100-180 net.

### Тестовое задание

1) Создать новый проект на Create React App с шаблоном typescript
2) Добавить зависимости @reduxjs/toolkit , react-router-dom@6
3) Создать две страницы News, NewsItem
4) Создать редюсер news для хранения новостей.
```
title: string;
author: string;
description: string;
url: string;
publishedAt: string;
content: string;
```
5) Получить API ключ на сайте https://newsapi.org/
6) На странице News вывести список новостей по теме (ключевому слову) education. Новость состоит из заголовка, даты, краткого описания и ссылки на страницу NewsItem (/news/:title).
5) На странице NewsItem, реализовать отображение выбранной новости. Заголовок, дата и полный текст (content). Также реализовать кнопку “Удалить”, которая будет удалять новость из хранилища и делать редирект на список новостей (без повторной загрузки новостей из API).

Выбор инструментов стилизации остается за исполнителем.
